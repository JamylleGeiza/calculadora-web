# Calculadora Web 🧮

Um projeto de calculadora interativa construída com as tecnologias base do desenvolvimento Web: HTML, CSS e JavaScript. Este projeto foi desenvolvido como uma prática de estruturação de interface e manipulação do DOM.

## 🚀 Funcionalidades

- **Operações Básicas:** Suporta adição, subtração, multiplicação e divisão.
- **Interface Responsiva:** Layout centralizado e organizado utilizando CSS Grid para a disposição dos botões.
- **Limpeza de Tela (C):** Zera completamente o visor.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estruturação semântica da página e dos elementos de input/botões.
- **CSS3:** Estilização visual da interface, com uso de `display: flex` para alinhamento central da página e `display: grid` para a matriz de botões. Efeitos de transição nos botões (`hover`).
- **JavaScript:** Lógica de programação para capturar os cliques (`onclick`), atualizar o visor em tempo real e processar as expressões matemáticas utilizando a função `eval()`.

## 📂 Como executar o projeto

1. Faça o clone deste repositório ou baixe os arquivos.
2. Abra a pasta do projeto no seu computador.
3. Dê um clique duplo no arquivo `index.html` para abri-lo no seu navegador padrão (Chrome, Firefox, Edge, etc.).
4. Não é necessário instalar nenhuma dependência ou servidor local para rodar.

## 💡 Aprendizados

Este projeto foi excelente para consolidar o fluxo de trabalho entre o design visual (idealizado previamente no Figma) e a tradução para código front-end. O maior foco foi entender a comunicação entre o HTML (estrutura) e o JavaScript (comportamento) através da manipulação de IDs e valores de inputs.
